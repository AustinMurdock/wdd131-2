Overview

Purpose

Allow users (particularly myself) to display and filter information about several pokemon at once, taking obtainability by generation and location for normal playthroughs as a major consideration for pokemon and their possible move coverages.

Audience

Myself and others who enjoy casual optimization for type coverage and stat distributions in standard pokemon playthroughs. This site should make it easier to find which pokemon are actually obtainable in a given game, and in what locations.

Dynamic elements

Content

By Location

The primary function of the site, to allow users to filter pokemon by proximity to in-game locations.
The specific structure will depend on whatever is most convenient to filter the PokeAPI location through, but it will likely include search/filter boxes for generation, location, level, type, or other desired parameters, and a map of available locations.

By Move Coverage

Pokemon can usually have only up to two types, but each can have several types of available moves depending on TM/HM availability and user preference, leading to a much wider type coverage. So this page would be similar to the location search page, but with a focus on typing and type coverage.
This section may be combined with the "By Location" page, depending on how the filtering methods work out. If so, I might just build a "generate random valid pokemon" function as a test.

Most information for either page will be sourced from PokeAPI, with some reference to Bulbapedia.
